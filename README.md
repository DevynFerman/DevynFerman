# Hey, I'm Dev 👋

Software Engineer at **The Kroger Co.** based out of Cincinnati, OH — working in the native iOS ecosystem that powers grocery shopping for millions of customers!

---

## What I'm working on 🛠️

My day-to-day sits at the intersection of iOS feature development and the tooling that keeps it all running. On the feature side, I'm building out the **Coupons, Weekly Ads, and Offers** experience in the Kroger iOS app. Previously on the platform side, I maintained the CI/CD infrastructure — GitHub Actions workflows, Mac mini runner farms, App Store Connect automation, and the release pipelines that ship a single iOS binary across every Kroger banner.

A few things I've been heads-down on lately:

- **App Store Connect Swift CLI** — automating screenshot uploads, certificate renewals, and provisioning profile management using a custom-built Swift CLI backed by the AppStoreConnect Swift SDK
- **Native iOS Statistics Reporting** — designing a Data Transformation Layer (DTL) on top of GitHub Actions to normalize build/test metrics into MongoDB, with Metabase dashboards for the team
- **Release automation** — cascade merge workflows, dynamic runner selection, and protected-branch push strategies for a multi-brand release cadence
- **iOS 26 Migration** - Converting some internal views and utilities to be Apple Glass'd for iOS 26!
